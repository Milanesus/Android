# Aplicaciones Android en Kotlin

Este repositorio contiene dos aplicaciones Android desarrolladas en Kotlin.

---

# Proyecto 1: Formulario de Intereses

## Carpeta: Ejercicio1

## Descripción:
Aplicación Android tipo formulario interactivo donde el usuario selecciona sus intereses personales mediante una interfaz visual.

La aplicación genera un resumen basado en las opciones seleccionadas.

## Funcionalidades:
- Selección de intereses mediante botones o imágenes
- Interfaz visual personalizada
- Navegación entre pantallas
- Generación de reporte final (Reporte.kt)

## Tecnologías usadas
- Kotlin
- Android Studio
- Diseño de interfaces con XML
- Navegación entre pantallas (Activities)

# Proyecto 2: Aplicación del Mundial 2026 utilizando una API de fútbol

## Carpeta: Ejercicio2

## Descripción:
Aplicación Android que consume una API de fútbol para mostrar información de equipos y jugadores del Mundial 2026.

## Arquitectura:
- Model - View - ViewModel (MVVM)

## Funcionalidades:
- Lista de equipos
- Lista de jugadores
- Navegación entre pantallas
- Manejo de estado con ViewModel

## Tecnologías usadas:
- Kotlin
- Android Studio
- Diseño de interfaces con XML
- Retrofit
- LiveData
